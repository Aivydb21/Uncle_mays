- **No African / Caribbean specialty importers.** Amaka's reference point is an African store in the south suburbs of Chicago. The Airtable does not contain that store or other Chicagoland ethnic specialty distributors. Worth a separate sourcing pass via Google Maps + cold visits.

- **No backup for Run A Way Buckers Club.** Pembroke is the current sole produce supplier; building a #2 in the same region (Pembroke is already Black-farmer territory) would be a defensive supply-chain move worth a separate research pass.
