## Trademark Registration Process

### Step 1: Comprehensive Search (Week 1)
- Hire trademark attorney to conduct full clearance search (federal, state, common law)
- Cost: $500-$1,500 per mark
- Timeline: 1-2 weeks

### Step 2: File Federal Applications (Week 2)
- **Filing basis:** Intent-to-Use (ITU) if store is not yet open, OR Use in Commerce if already operating
- **Classes:** File in Classes 35, 29, 30 at minimum (3 classes = $350 x 3 = $1,050 in USPTO fees)
- **Filing type:** TEAS Plus (cheaper) or TEAS Standard
- **Application components:**
  - Mark: "Uncle May's" (word mark) and "Uncle May's Produce" (word mark)
  - Logo: Uncle May's logo (design mark, if applicable)
  - Specimen of use: Photos of storefront signage, packaging, website (if Use in Commerce)
  - Statement of use: Description of goods/services in each class

**Cost:**
- USPTO filing fees: $350 per class (TEAS Plus) or $750 per class (TEAS Standard)
- Attorney fees (if using counsel): $1,500-$3,000 per mark
- **Total estimated cost (DIY, 3 classes):** $1,050
- **Total estimated cost (with attorney, 3 classes):** $4,500-$10,000

### Step 3: USPTO Examination (Months 3-6)
- USPTO examining attorney reviews application
- May issue **Office Action** (objection or refusal) requiring response
- Attorney responds to Office Action (additional $500-$1,500 in legal fees if needed)

### Step 4: Publication for Opposition (Months 6-9)
- If no objections, mark is **published in the Official Gazette**
- 30-day opposition period (third parties can oppose registration)
- If no opposition filed, proceed to registration

### Step 5: Statement of Use (Months 9-12, if ITU filing)
- If filed on Intent-to-Use basis, must file **Statement of Use** showing actual use in commerce
- Submit photos of storefront, packaging, website showing the mark in use
- Fee: $100 per class
- Deadline: 6 months after Notice of Allowance (can extend up to 3 years for $150 per extension)

### Step 6: Registration (Months 10-14)
- USPTO issues **Certificate of Registration**
- Mark is officially registered (® symbol can be used)
- Registration is valid for **10 years** (renewable indefinitely)

## Risks of NOT Registering

### 1. Competitor Infringement
- Another grocery store (or food brand) could start using "Uncle May's" or a confusingly similar name
- Without federal registration, enforcement is difficult and expensive (requires proving common law rights)
- **Example:** If a competitor opens "Uncle Mike's Produce" in Los Angeles, we may have no legal recourse without a registered trademark

### 2. Loss of Trademark Rights
- If a competitor files a federal trademark application for "Uncle May's" **before we do**, they could block our registration and potentially force us to rebrand
- **First-to-file** system in the U.S. (not first-to-use) — whoever files first has priority, even if we used the name first

### 3. Investor Concerns
- VCs and strategic acquirers conduct IP due diligence
- Lack of trademark protection signals weak IP foundation and increases risk premium
- Could reduce valuation by 10-20% in future fundraising or exit negotiations

### 4. Brand Dilution
- Without ®, the brand is weaker and more vulnerable to copycats, parody, or dilution by unrelated businesses using "Uncle May's"

### 5. Franchise / Licensing Limitations
- Cannot franchise or license the "Uncle May's" brand to third parties without a registered trademark
- Limits future revenue opportunities (e.g., licensing Uncle May's private label to other retailers)
