## Usage Instructions

### For Print Materials
1. **Flyers (8.5" x 11"):** Place QR code in bottom-right corner, minimum 1.5" x 1.5" size
2. **Door Hangers (4.25" x 11"):** Place QR code above tear-off coupon, minimum 1.25" x 1.25" size
3. **Posters:** Scale up as needed, QR code is vector-quality at high DPI

### Minimum Scan Size
- **Physical size:** 1" x 1" minimum for reliable smartphone scanning
- **Recommended:** 1.5" x 1.5" or larger for ease of use
- **Distance:** Scannable from 6-12 inches with most smartphones

### Testing
Test the QR code before printing by:
1. Opening `qr-code-neighbor20.png` on a computer screen
2. Scanning with iPhone or Android camera app
3. Verifying it redirects to: `https://unclemays.com/?utm_source=qr-code&utm_medium=offline&utm_campaign=hyde-park-local&promo=NEIGHBOR20`
4. Checking that NEIGHBOR20 promo code is visible/applied on checkout page
