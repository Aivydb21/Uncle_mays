## Generation Settings

**Common parameters across all prompts:**
- `--v 6` — Midjourney version 6 (latest, best quality)
- `--style raw` — Photographic realism (not artistic/painterly)
- `--ar` — Aspect ratio (1:1 square, 4:5 portrait, 16:9 landscape)

**Quality tips:**
- Always use highest resolution available
- Select "Upscale" (U1-U4) on best variant before downloading
- If first result isn't perfect, regenerate with slight prompt tweaks
