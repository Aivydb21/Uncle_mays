## Design Notes

### Typography
- **Headlines:** Bold, sans-serif (Montserrat, Poppins, or similar)
- **Body/Subheads:** Medium weight, clean sans-serif
- **CTAs:** Bold, high contrast, button-style backgrounds

### Color Palette
- **Primary:** Green (#2D5016 or similar earth tone, matches produce/freshness)
- **Accent:** Gold/Yellow (#D4AF37 for cultural warmth)
- **Background overlays:** Dark semi-transparent (rgba(0,0,0,0.4)) for text readability over photos
- **CTA buttons:** Bright green (#4CAF50) or gold
- **Promo badges:** Vibrant green (#00C853) for urgency

### Logo Placement
- Always include Uncle May's logo
- Size: 80-120px height depending on canvas size
- Placement: Top left/center/right or bottom (varies by layout)
- Ensure logo doesn't obstruct key visual elements

### Text Readability
- Use dark overlays on busy background images
- Minimum font size: 24pt (for smallest text elements)
- High contrast: white text on dark overlay, or dark text on light badge
- Avoid text over cluttered areas of product photos

1. Open Canva, create custom size design for each format
2. Upload product photos (`heritage.jpg`, `hero-produce.jpg`, `produce-box.jpg`, `uncle-mays-logo.png`)
3. Follow specifications above for each design
4. Export as PNG (high quality)
5. Save to `ad-exports/subscription-launch-apr17/static-images/`

### Option 2: Canva API (Automated, if time permits)
1. Upload assets via `/assets/upload` endpoint
2. Create designs via `/designs` endpoint with presets
3. Modify via Canva's design API (if supported for text overlays)
4. Export via `/exports` endpoint to PNG
5. Download and save locally
