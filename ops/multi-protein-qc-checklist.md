## Cold-Chain Safety Protocol

### Pre-Packing Checks
- [ ] All proteins stored at ≤40°F before packing
- [ ] Cooler temp recorded on Cold Chain Log: ______°F at ______ AM/PM
- [ ] Ice packs prepared: _____ packs (2-4 depending on protein count)

### Cross-Contamination Prevention
- [ ] Hands washed between handling different protein types
- [ ] Chicken and pork double-bagged in leak-proof bags
- [ ] Salmon in leak-proof container (skin-side down)
- [ ] Beef in vacuum-sealed package
- [ ] Separate handling zones used (Chicken → Pork → Beef → Fish order)

### Packing Order (Bottom to Top)
1. [ ] Ice packs (bottom layer)
2. [ ] Beef (if included)
3. [ ] Pork (if included)
4. [ ] Salmon (if included)
5. [ ] Chicken (if included, top layer)
6. [ ] Produce items packed around/above proteins
